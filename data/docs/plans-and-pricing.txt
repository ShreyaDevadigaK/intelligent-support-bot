PLAN CHANGES

Upgrading:
- Instant access to new features
- Pro-rated charge for remaining days in cycle
- No downtime during upgrade

Downgrading:
- Takes effect at next billing cycle
- Unused credit applied to next invoice (not refunded as cash)
- If current usage exceeds new plan limits, must reduce usage first
  (e.g., reduce users to under new plan limit before downgrading)
